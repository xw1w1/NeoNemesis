INTEGRATION NOTES:
===================

To use in other parts of Nemesis:
  1. Include <System API Rendering/SystemShaderBloom/SystemShaderBloom.hpp>
  2. Create with: CreateBloomEffect(device, context, width, height)
  3. Render with: bloom->Render(sourceTexture, color, params)
  4. Get output: bloom->GetOutputTexture()

Thread Safety:
  - Must be used on render thread
  - Device/context should be from single thread
  - No concurrent rendering supported

Memory Requirements:
  - Minimum: ~3 MB (0.5 downscale, 1920x1080)
  - Maximum: ~12 MB (full res, 4 targets)
  - Scales linearly with resolution

GPU Bandwidth (per frame):
  - Typical: 50-100 MB at 60 FPS
  - With downscaling: 12-25 MB (ideal)
